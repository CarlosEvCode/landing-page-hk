# Hollow Knight Landing Page

Landing page minimalista inspirada en el universo de Hollow Knight.

## Despliegue
El proyecto está alojado en: https://hollowknight-cm.netlify.app/

## Tecnologías
- HTML5 semántico
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Vanilla JS)
- Google Material Symbols
- Google Fonts (Inter)

## Características
- Estructura de 7 secciones
- Diseño responsivo y optimizado
- Navegación con scroll suave
- Formulario de contacto con validación nativa
- Galería de activos visuales oficiales

## Ejecución
Para visualizar el proyecto localmente:
1. Clonar el repositorio.
2. Abrir index.html en un navegador o usar un servidor local (ej: python -m http.server).
